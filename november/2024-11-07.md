### 🚀 Today I Learned

#### | 리눅스
mdadm





RAID 구성 명령어



관련 정보





/proc/mdstat



파티션 정보: /proc/partitions



RAID





0 : 스트라이핑 / 2개 디스크



1 : 미러링



5 : 3개 디스크



6 : 4개 디스크



10 : 1+0, 4개 디스크



명령어





-C : 생성



-l : 레벨 지정



-n : 디바이스 개수



-D : 자세한 정보



-S : 전부 비활성화



-r : 제거



-a : 작동중인 RAID에 디스크 추가



-f : 고의 오류 유발



명령어 구성





mdadm 대문자 파일 레벨 드라이브 파일
